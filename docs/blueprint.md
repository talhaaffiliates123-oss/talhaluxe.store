# **App Name**: Talha Luxe

## Core Features:

- Product Catalog: Display all available products, including images, descriptions, and prices.
- Shopping Cart: Enable users to add products to a cart, update quantities, and calculate total cost.
- Secure Checkout: Allow users to securely enter shipping and payment information to complete their purchase using Stripe.
- Order Management: Provide a system for admins to add, edit, and delete products, manage orders, and track payments via Firebase.
- User Authentication: Allow users to create accounts, login, and view their order history using Firebase Authentication.
- Product Recommendation Tool: Suggest related items to users based on browsing history and best selling products; this feature uses an AI tool to identify these correlations.
- Wishlist: Allows user to add a product to a personalized wishlist

## Style Guidelines:

- Primary color: Deep charcoal (#343A40) for a premium, luxury feel.
- Background color: Soft gray (#F8F9FA) to provide a clean and modern backdrop.
- Accent color: Pale gold (#D4AF37) to highlight key elements and call-to-action buttons.
- Body and headline font: 'Montserrat', a sans-serif font, is ideal for clear product descriptions and titles. Note: currently only Google Fonts are supported.
- Use professional, minimalist icons for navigation and key features.
- Implement a clean, modern layout with a sticky navigation bar and clear product cards.
- Incorporate smooth hover effects and subtle animations to enhance user engagement.
